RhythmGame Readme

■このゲームについて
　□キーボード又はジョイスティックで遊ぶことができるリズムゲームです。
　□このプログラムは現在α版です。意図しない不具合が発生する可能性がないとは言えませんので同意の上実行してください。

■動作環境
　□必須環境
　　・Windows OSの動作しているPC
　　・Microsoft Visual C++ 2015 x86 Redistributableがインストールされていること
　　・インストールされていない方はhttps://www.microsoft.com/ja-jp/download/details.aspx?id=48145からvc_redist.x86.exeをダウンロード、インストールしてください。
　□テスト環境1
　　・OS          : Windows 10 Pro 64bit
　　・CPU         : Intel(R) Core(TM) i7-6700K CPU@4.00GHz 4.01GHz
　　・物理メモリ  : 16.00GB
　□テスト環境2
　　・OS          : Windows 10 Pro 64bit
　　・CPU         : Intel(R) Core(TM) i7-3667U CPU@2.00GHz 2.00GHz
　　・物理メモリ  : 8.00GB

■ファイル構成
　□START.exe : ゲーム本体
　□Readme.txt : このファイル
　□Data : リソースフォルダ
　　□bgm : BGMフォルダ
　　□img : 画像フォルダ
　　□sound-effect : SEフォルダ
　　□songs : 楽曲フォルダ
　　□conf.ini : 設定ファイル
　　□savedata.dat : セーブデータ

■操作方法
　□楽曲プレイ画面(デフォルト値)
　　・真上から来るノーツ――――(キーボード)W・I  , (ジョイスティック)↑・ボタン3
　　・真下から来るノーツ――――(キーボード)S・K  , (ジョイスティック)↓・ボタン2
　　・右から来るノーツ―――――(キーボード)D・L  , (ジョイスティック)→・ボタン1
　　・左から来るノーツ―――――(キーボード)A・J  , (ジョイスティック)←・ボタン4
　　・右上から来るノーツ――――(キーボード)O     , (ジョイスティック)ボタン6
　　・左上から来るノーツ――――(キーボード)Q     , (ジョイスティック)ボタン5
　　・ポーズ――――――――――(キーボード)F1・F2, (ジョイスティック)ボタン7・ボタン8
　　・ポーズから復帰――――――(キーボード)F1＋F2, (ジョイスティック)ボタン7＋ボタン8
　　・復帰中に再度ポーズ――――(キーボード)O＋Q  , (ジョイスティック)ボタン5＋ボタン6
　　・終了―――――――――――(キーボード)ESC

　□その他の画面(デフォルト値)
　　・↑――――――――――――(キーボード)W     , (ジョイスティック)↑
　　・↓――――――――――――(キーボード)S     , (ジョイスティック)↓
　　・→――――――――――――(キーボード)D     , (ジョイスティック)→
　　・←――――――――――――(キーボード)A     , (ジョイスティック)←
　　・決定―――――――――――(キーボード)L     , (ジョイスティック)ボタン1
　　・キャンセル――――――――(キーボード)K     , (ジョイスティック)ボタン2
　　・終了―――――――――――(キーボード)ESC
